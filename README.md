# SkillMap AI 🚀

SkillMap AI is a simple web-based tool that analyzes user skills and compares them with industry requirements for a selected job role.

## 🔥 Features
- Skill matching
- Match score calculation
- Missing skill detection
- Simple career suggestions

## ⚙️ Tech Stack
- HTML
- CSS
- JavaScript

## 🧠 How It Works
1. User selects a role
2. Enters skills
3. System compares with dataset
4. Displays match score and missing skills
